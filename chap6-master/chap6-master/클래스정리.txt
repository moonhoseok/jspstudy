추상화 => 클래스 생성
          변수 - 클래스변수    :  클래스정보 로드시 변수(메모리할당).
                    인스턴스변수 :  객체화시 변수(메모리 할당)

          메서드 - 클래스 메서드 : 클래스 정보 로드시 메서드
                    - 인스턴스 메서드(생성자) : 객체화시  메서드

          초기화블럭 - static초기화 블럭 : 클래스 정보 로드시 한번 실행
                                                           클래스변수 초기화
                            인스턴스초기화블럭 : 객체화 마다 실행.
                                 인스턴스변수의 초기화.(생성자와 기능겹침)
       
      this : 자기 참조변수
      this 생성자 : 같은 클래스 내의 다른 생성자 호출 

           오버로딩 : 같은 클래스내에 이름 같고, 매개변수가 다른 메서드가 존재.
